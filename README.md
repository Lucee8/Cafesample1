☕ Cafe Website

A modern and responsive cafe website designed to showcase menu items, attract customers, and provide seamless reservation and contact options.

🚀 Features

🌟 Hero Section with Rating
Eye-catching cafe imagery with a 4.8★ rating display

🏠 About Section
Highlights with real stats and cafe details

🍔 Menu Section
6 categories: Burgers, Pizza, Pasta, Sandwiches, Drinks, Desserts with realistic pricing

👨‍🍳 Chef Specials
Featured dishes with high-quality food images

🖼️ Photo Gallery
Cafe atmosphere and food images

💬 Customer Reviews
Guest testimonials inspired by real feedback

📍 Visit Us Section
Address with clickable Google Maps integration

📅 Table Reservation Form
Easy booking with contact details

📱 WhatsApp Integration
Floating button for quick communication

📲 Responsive Design
Mobile-friendly layout with hamburger menu

🛠️ Technologies Used

HTML5

CSS3

JavaScript
